# Dio English Tutor — Web Platform

Full-stack web platform for learning English (CEFR A1–C2 + IELTS preparation).

## Stack
- **Backend:** Django 5 + DRF + SimpleJWT + PostgreSQL
- **Frontend:** Next.js 15 + TypeScript + Tailwind CSS
- **Auth:** JWT (access + refresh, role-based: student / teacher / admin)
